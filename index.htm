Motiviation for ProjectA Magical Car

We hoped to create a magical experiance, drawing form the Harry Potter universe. Our past VR inspiration is for the numerous "dangerous" driving simulators that exist. In our game, not only does the user do things that are quite dangerous in real life, but many things which are outright impossible with current techinology, such as having a car fly. VR is an apt medium to express this, and allow the user to enjoy these things they could not do in real life. Furthermore, we wanted engross the user in a familiar world, that of Harry Potter, which most likely, also does not exist in real life. We hoped to create an experience that could only be done in VR, to allow one to feel like they are driving, and flying, in a magical car, with Harry Potter.

ExplanationOur Project

The main technology we used to create our experience was Google cardboard, which we developed for with the Unity editor. We used head tracking to allow the user to look around, and the built in stereoscopic 3d to enhance the experience. We did not have a conveniant way for the user to drive the car in a life like manner, and had to resort to using a simple controller for driving/flying. However, we were able to use a tactile feedback vest to improve the experience, and allow the user to feel as if they are in the car, and to feel the engine sounds, and bumps.
